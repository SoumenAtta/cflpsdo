The instances for the Capacitated Facility Location Problem with Service Distance and Outsourcing (CFLPSDO) mentioned in this page are generated using the maximal covering location problem (MCLP) [1,2,3] instances which are available at http://www.lac.inpe.br/~lorena/instancias.html. We have created 45 CFLPSDO instances. The size of an instance is determined by the number of customers or potential facility locations (n). Each instance of CFLPSDO consists of distance matrix having size n × n, demand vector of size n × 1, capacity matrix having size n × n, and external cost matrix having size n × 1.
